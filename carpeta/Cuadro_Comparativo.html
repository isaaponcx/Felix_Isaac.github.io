Cuadro comparativo HTML y HTML5

Característica HTML HTML5
Doctype <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> <!DOCTYPE html>
Estructura semántica Uso de divs y etiquetas básicas; pocas etiquetas semánticas propias. Nuevos elementos semánticos: header, nav, section, article, aside, footer, main, etc.
Multimedia y gráficos nativos Sin soporte nativo para video/audio/Canvas; se usan plugins para multimedia. Soporte nativo para video, audio y canvas; gráficos con canvas; video/audio integrados.
Formularios y validación Campos básicos; validación limitada; no hay nuevos tipos de input por estándar. Nuevos tipos de input (email, url, date, number, color, etc.), placeholders, required, pattern, autocomplete, etc.
Compatibilidad y enfoque Uso de formatos y atributos heredados; compatibilidad con navegadores antiguos. Mejor compatibilidad en navegadores modernos; enfoque en estructuras y APIs modernas para web apps.
